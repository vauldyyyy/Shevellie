# Shevellie & Vauldan

A special application built with love.

## Project info

This project is a React application built with Vite.

## Getting Started

To run this project locally:

1.  Clone the repository
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
